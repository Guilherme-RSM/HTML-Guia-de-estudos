HTML - Guia de Estudos

Este Guia de estudos é um compilado resumido e simplificado de artigos da Web para facilitar a compreensão e aprendizado do HTML5.

O que é HTML?

HTML = Hyper Text Marckup Language. Que significa Linguagem de Marcação de HyperTexto.

Portanto o HTML não é uma linguagem de programação e sim uma linguagem de marcação.

Para que serve o HTML?

Essa linguagem de marcação é utilizada na construção da estrutura das páginas da Web.

Origem do HTML

O HTML foi criado por Tim Berners-Lee, em 1991 no CERN (Conselho Europeu para Pesquisa Nuclear).
